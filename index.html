About Me

Systems Engineer with a track record of building mission-critical platforms where uptime and data integrity are non-negotiable.

I specialize in DevOps and SRE practices for enterprise-scale healthcare and financial platforms — managing API gateways, container orchestration, and cloud-native pipelines that collectively process over 1 million API calls daily. I'm on-call when things break, and I build the systems that prevent them from breaking in the first place.
